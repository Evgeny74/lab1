Лабораторная работа №1

Реализована структура классов в сфере автомобилестроение.

Создан интерфейс автомобиля (ICar) c методами:
void move - метод движения;
void stop - метод остановки;
void overtake - метод обгона другого автомобиля;
void OpenDoors - метод открытия дверей;
void refuel - метод заправки автомобиля.
Поля:
Name - имя автомобиля;
TypeOfFuel - тип используемого топлива;
SizeOfFuelTank - размер топливного бака;
moving - поле, в котором показывает, движется автомоблиь или нет;
FuelOfThisCar - поле топлива этого автомобиля.

Реализован абстрактный класс автомобиль Automobile, являющийся наследником ICar. Внем реализованы методы:
move - метод движения;
stop - метод остановки;
overtake - метод обгона другого автомобиля;
OpenDoors - метод открытия дверей;
refuel - метод заправки автомобиля.
Поля:
Name - имя автомобиля;
TypeOfFuel - тип используемого топлива;
SizeOfFuelTank - размер топливного бака;
moving - поле, в котором показывает, движется автомоблиь или нет;
FuelOfThisCar - поле топлива этого автомобиля.

Классы Car, Truck и Lorry - наследники класс Automobile.

Truck - класс грузового автомобиля. Помимо унаследованных от Automobile методов обладает методом beloaded - метод загрузки грузовика.

Lorry - класс тягача (фуры). Помимо унаследованных от Automobile методов обладает методом AttachATrailer - метод прицепления прицепа ITrailer.

Car - класс легкового автомобиля. Помимо унаследованных от Automobile методов обладает методом openBoot - метод открытия багажника.

Классы SportsCar и OffRoader являются наследниками класса Car. Методы и поля те же, что и у класса Car.

Был реализован класс заправка FillingStation для заправки автомобилей.
Методы:
fillTheCar - метод заправки автомобиля.
Поля:
Petrol_98 p98 - 98-ой бензин;
Petrol_95 p95 - 95-ый бензин.

Был создан интерфейс IFuel (топливо) с 2 полями: Type (95-ый или 98-ой бензин) и FuelLeft - количество оставшегося топлива.

Были реализованы классы Petrol95 и Petrol98 (95-ый и 98-ой бензин), которые являются наследниками IFuel.

Был создан интерфейс трейлера ITrailer с методом BeAttached и соответсвующий класс Trailer, являющийся его наследниокм.
